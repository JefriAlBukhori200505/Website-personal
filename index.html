📘 Mengenal HTML, CSS, JS, dan Tailwind

HTML

Struktur dasar halaman web, seperti teks, gambar, dan tautan.

Contoh: <h1>Halo Dunia</h1>

CSS

Mengatur tampilan web agar lebih indah, dengan warna dan layout.

Contoh: body { color: blue; }

JavaScript

Menambah interaktivitas seperti tombol, efek, dan animasi.

Contoh: alert('Halo Dunia!');

Tailwind CSS

Framework CSS cepat dengan class siap pakai yang efisien.

Contoh: <div class="text-red-500">Hello</div>

💡 Langkah Membuat Website Sederhana

  1. Buat file index.html, style.css, dan script.js
  2. Tulis struktur HTML
  3. Tata desain dengan CSS / Tailwind
  4. Tambahkan efek interaktif pakai JavaScript
